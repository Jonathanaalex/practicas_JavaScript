¿Qué son los módulos en JavaScript?

Los módulos en JavaScript permiten dividir el código en archivos separados, lo que facilita el mantenimiento, la reutilización y la organización del código. Con los módulos, puedes exportar funciones, objetos o valores primitivos desde un archivo e importarlos en otro.

Ejercicio 1: Módulos Básicos

Este ejercicio demuestra cómo crear y usar módulos básicos con exportaciones nombradas y por defecto.

miproyecto/
│
├── index.html
├── main.js
├── saludo.js
├── despedida.js
└── mensaje.js

saludo.js

// Exportación nombrada export const saludo = () => { return "¡Hola, Mundo!"; };

despedida.js

// Exportación nombrada export const despedida = () => { return "¡Adiós, Mundo!"; };

mensaje.js

// Exportación por defecto const mensaje = "Mensaje por defecto"; export default mensaje;

main.js

// Importar las funciones y el mensaje import { saludo } from './saludo.js'; import { despedida } from './despedida.js'; import mensaje from './mensaje.js'; // Ejecutar las funciones y mostrar resultados console.log(saludo()); console.log(despedida()); console.log(mensaje);

Ejercicio 2: Módulo de Operaciones Matemáticas

Este ejercicio muestra cómo crear un módulo con funciones matemáticas y cómo importarlas.

math.js

// math.js - Módulo de operaciones matemáticas export const suma = (a, b) => a + b; export const resta = (a, b) => a - b; export const multiplicacion = (a, b) => a * b; export const division = (a, b) => a / b;

Uso en main.js

import { suma, resta, multiplicacion, division } from './math.js'; console.log("Suma: 2 + 3 = " + suma(2, 3)); console.log("Resta: 5 - 2 = " + resta(5, 2)); console.log("Multiplicación: 4 * 6 = " + multiplicacion(4, 6)); console.log("División: 15 / 3 = " + division(15, 3));

Ejercicio 3: Módulos Adicionales

En este ejercicio, creamos módulos adicionales para manipulación de cadenas y operaciones con arrays.

stringOperations.js

// stringOperations.js - Módulo para manipulación de cadenas export const capitalizar = (texto) => { return texto.charAt(0).toUpperCase() + texto.slice(1); }; export const invertirCadena = (texto) => { return texto.split('').reverse().join(''); }; export const contarCaracteres = (texto) => { return texto.length; };

arrayUtilities.js

// arrayUtilities.js - Módulo para operaciones con arrays export const encontrarMaximo = (array) => { return Math.max(...array); }; export const encontrarMinimo = (array) => { return Math.min(...array); }; export const sumarArray = (array) => { return array.reduce((total, num) => total + num, 0); }; export const filtrarPares = (array) => { return array.filter(num => num % 2 === 0); };

Uso de los módulos

import { capitalizar, invertirCadena } from './stringOperations.js'; import { encontrarMaximo, sumarArray } from './arrayUtilities.js'; console.log("Capitalizar 'javascript': " + capitalizar("javascript")); console.log("Invertir 'hola': " + invertirCadena("hola")); const numeros = [1, 5, 3, 8, 2]; console.log("Array: " + numeros); console.log("Máximo: " + encontrarMaximo(numeros)); console.log("Suma: " + sumarArray(numeros));

Ejercicio 4: Combinación de Todos los Módulos

Este ejercicio combina todos los módulos anteriores en un solo archivo principal.

main-completo.js

// Importar todos los módulos import { saludo } from './saludo.js'; import { despedida } from './despedida.js'; import mensaje from './mensaje.js'; import { suma, resta, multiplicacion, division } from './math.js'; import { capitalizar, invertirCadena } from './stringOperations.js'; import { encontrarMaximo, sumarArray } from './arrayUtilities.js'; // Ejecutar funciones de todos los módulos console.log("=== MÓDULOS BÁSICOS ==="); console.log(saludo()); console.log(despedida()); console.log(mensaje); console.log("\n=== OPERACIONES MATEMÁTICAS ==="); console.log("5 + 3 = " + suma(5, 3)); console.log("10 - 4 = " + resta(10, 4)); console.log("6 * 7 = " + multiplicacion(6, 7)); console.log("20 / 5 = " + division(20, 5)); console.log("\n=== OPERACIONES CON CADENAS ==="); console.log("Capitalizar 'ejemplo': " + capitalizar("ejemplo")); console.log("Invertir 'javascript': " + invertirCadena("javascript")); console.log("\n=== OPERACIONES CON ARRAYS ==="); const numeros = [12, 5, 23, 8, 17]; console.log("Array: " + numeros); console.log("Máximo: " + encontrarMaximo(numeros)); console.log("Suma total: " + sumarArray(numeros));
