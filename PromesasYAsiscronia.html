Ejercicio 1: Callback de suma

Este ejercicio muestra el uso de callbacks para manejar operaciones asíncronas. La función sumar recibe dos números y un callback que se ejecuta con el resultado.

Código JavaScript

function sumar(a, b, callback) {
    const resultado = a + b;
    callback(resultado);
}

function mostrarResultado(resultado) {
    document.getElementById('resultado').innerHTML = resultado;
}

sumar(5, 10, mostrarResultado);

Ejercicio 2: Callback con setTimeout

Este ejercicio simula una operación asíncrona con setTimeout. Después de 2 segundos, se ejecuta el callback con los datos obtenidos.

Código JavaScript

function obtenerDatos(callback) {
    setTimeout(() => {
        const datos = { nombre: "Juan", edad: 30 };
        callback(datos);
    }, 2000);
}

function mostrarDatos(datos) {
    document.getElementById('resultado').innerHTML = 
        `Nombre: ${datos.nombre}, Edad: ${datos.edad}`;
}

obtenerDatos(mostrarDatos);

Ejercicio 3: Promesas con API

Este ejercicio utiliza promesas para obtener datos de una API (JSONPlaceholder). Se usa fetch para hacer la solicitud y then/catch para manejar la respuesta y los errores.

Código JavaScript

function obtenerPosts() {
    fetch('https://jsonplaceholder.typicode.com/posts')
        .then(response => {
            if (!response.ok) {
                throw new Error('Error en la respuesta de la API');
            }
            return response.json();
        })
        .then(data => {
            mostrarPosts(data);
        })
        .catch(error => {
            console.log('Error:', error);
        });
}

Ejercicio 4: Async/Await con API

Este ejercicio hace lo mismo que el anterior pero usando async/await, que permite escribir código asíncrono de manera más clara y estructurada.

Código JavaScript

async function obtenerPostsConAsyncAwait() {
    try {
        let response = await fetch(
            'https://jsonplaceholder.typicode.com/posts'
        );
        if (!response.ok) {
            throw new Error('Error en la llamada a la API');
        }
        let posts = await response.json();
        mostrarPosts(posts);
    } catch (error) {
        console.log('Error:', error);
    }
}
