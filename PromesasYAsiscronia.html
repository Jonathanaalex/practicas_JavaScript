Ejercicio 4: Mayor de 2 números

Crea una función tradicional y una función flecha para calcular el mayor de dos números.

Función Tradicional

    if (num1 > num2) {
        return num1;
    } else {
        return num2;
    }
}

Función Flecha

const mayorFlecha = (num1, num2) => 
    (num1 > num2) ? num1 : num2;

Ejercicio 7: Closure para contar

Este ejemplo muestra cómo usar closures para crear un contador con estado privado.

Función Principal

function crearContador() {
    let contador = 0;
    return function() {
        contador += 1;
        return contador;
    };
}

Explicación

La función crearContador define una variable contador que no es accesible desde el exterior. Sin embargo, la función interna que se devuelve puede acceder y modificar contador, creando un estado privado.

Ejercicio 8: Función que retorna otra función

Este ejercicio demuestra cómo crear una función que devuelve otra función, ilustrando el concepto de closures en JavaScript.

Función Principal

function crearSaludo(saludo) {
    return function(nombre) {
        return `${saludo}, ${nombre}!`;
    };
}

Creación de Funciones

const saludarEnEspanol = crearSaludo("Hola");
const saludarEnIngles = crearSaludo("Hello");
const saludarEnFrances = crearSaludo("Bonjour");

Ejercicio 10: Juego de adivinar el número

Un juego interactivo que genera un número aleatorio y permite al usuario adivinarlo.

Función Principal

function iniciarJuegoAdivinanza(min, max) {
    const numeroAleatorio = Math.floor(Math.random() * (max - min + 1)) + min;
    let intentos = 0;
    
    return function(adivinanza) {
        intentos++;
        if (adivinanza === numeroAleatorio) {
            return `¡Correcto! Adivinaste en ${intentos} intentos.`;
        } else if (adivinanza < numeroAleatorio) {
            return "Demasiado bajo. Intenta de nuevo.";
        } else {
            return "Demasiado alto. Intenta de nuevo.";
        }
    };
}

Implementación

Esta función utiliza closures para mantener el estado del juego (número aleatorio y contador de intentos) entre llamadas.
