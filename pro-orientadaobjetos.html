Clases y Objetos

¿Qué son las Clases y Objetos?

En la Programación Orientada a Objetos, una clase es una plantilla para crear objetos. Define un tipo de objeto según un conjunto de propiedades (atributos) y métodos (funciones).

Un objeto es una instancia de una clase. Es una entidad que tiene un estado (datos) y comportamiento (métodos) definidos por la clase.

En JavaScript, las clases se introdujeron en ECMAScript 2015 (ES6) y proporcionan una sintaxis más clara para la creación de objetos y la herencia.

Ejemplo de Código

// Definición de una clase
class Animal {
    constructor(nombre, sonido) {
        this.nombre = nombre;
        this.sonido = sonido;
    }
    
    hacerSonido() {
        return `${this.nombre} hace ${this.sonido}`;
    }
}

// Creación de objetos (instancias)
const perro = new Animal('Perro', 'guau');
const gato = new Animal('Gato', 'miau');

console.log(perro.hacerSonido()); // "Perro hace guau"
console.log(gato.hacerSonido());  // "Gato hace miau"

Herencia

¿Qué es la Herencia?

La herencia es un mecanismo que permite crear una nueva clase basada en una clase existente. La clase nueva (subclase) hereda las propiedades y métodos de la clase base (superclase).

Esto permite la reutilización de código y la creación de jerarquías de clases donde las clases más específicas extienden el comportamiento de las más generales.

En JavaScript, se utiliza la palabra clave extends para crear una subclase, y super() para llamar al constructor de la clase padre.

Ejemplo de Código

// Clase base
class Animal {
    constructor(nombre, sonido) {
        this.nombre = nombre;
        this.sonido = sonido;
    }
    
    hacerSonido() {
        return `${this.nombre} hace ${this.sonido}`;
    }
}

// Subclase que hereda de Animal
class Perro extends Animal {
    constructor(nombre, raza) {
        super(nombre, 'guau'); // Llama al constructor de la clase base
        this.raza = raza;
    }
    
    mostrarRaza() {
        return `${this.nombre} es un ${this.raza}`;
    }
}

// Creación de objeto de la subclase
const miPerro = new Perro("Max", "Labrador");
console.log(miPerro.hacerSonido()); // "Max hace guau"
console.log(miPerro.mostrarRaza()); // "Max es un Labrador"

Polimorfismo

¿Qué es el Polimorfismo?

El polimorfismo permite que diferentes clases se traten como si fueran de una clase común. Es la capacidad de un método de hacer cosas diferentes en función del objeto que lo llama.

En la práctica, esto significa que podemos definir un método en una clase base y luego sobrescribirlo en las subclases para proporcionar un comportamiento específico.

Esto nos permite trabajar con objetos de diferentes tipos de manera uniforme, llamando al mismo método en todos ellos pero obteniendo resultados diferentes según el tipo específico de cada objeto.

Ejemplo de Código

class Animal {
    constructor(nombre, sonido) {
        this.nombre = nombre;
        this.sonido = sonido;
    }
    
    hacerSonido() {
        return `${this.nombre} hace ${this.sonido}`;
    }
}

class Perro extends Animal {
    constructor(nombre, raza) {
        super(nombre, 'guau');
        this.raza = raza;
    }
    
    // Sobrescribir el método hacerSonido
    hacerSonido() {
        return `${this.nombre} ladra: ${this.sonido}`;
    }
}

class Gato extends Animal {
    constructor(nombre) {
        super(nombre, 'miau');
    }
    
    // Sobrescribir el método hacerSonido
    hacerSonido() {
        return `${this.nombre} maúlla: ${this.sonido}`;
    }
}

// Crear array de diferentes animales
const animales = [new Perro('Max', 'Labrador'), new Gato('Luna')];

// Cada animal responde de manera diferente al mismo método
animales.forEach(animal => {
    console.log(animal.hacerSonido());
});

Ejercicio Vehículo

Instrucciones del Ejercicio

1. Crear una clase base Vehículo: con propiedades marca y modelo, y un método mostrarInfo().

2. Crear una subclase Coche: que herede de Vehículo y añada la propiedad puertas y el método mostrarPuertas().

3. Crear instancias de ambas clases y utilizar sus métodos.

4. Agregar más Subclases: como Moto o Camion con propiedades y métodos únicos.

5. Agregar al menos otro método que consideres necesario.

Solución del Ejercicio

class Vehiculo {
    constructor(marca, modelo) {
        this.marca = marca;
        this.modelo = modelo;
    }
    
    mostrarInfo() {
        return `${this.marca} ${this.modelo}`;
    }
    
    tipoVehiculo() {
        return "Vehículo genérico";
    }
}

class Coche extends Vehiculo {
    constructor(marca, modelo, puertas) {
        super(marca, modelo);
        this.puertas = puertas;
    }
    
    mostrarPuertas() {
        return `Este coche tiene ${this.puertas} puertas`;
    }
    
    tipoVehiculo() {
        return "Coche";
    }
}

class Moto extends Vehiculo {
    constructor(marca, modelo, cilindrada) {
        super(marca, modelo);
        this.cilindrada = cilindrada;
    }
    
    mostrarCilindrada() {
        return `Cilindrada: ${this.cilindrada}cc`;
    }
    
    tipoVehiculo() {
        return "Moto";
    }
}

class Camion extends Vehiculo {
    constructor(marca, modelo, capacidadCarga) {
        super(marca, modelo);
        this.capacidadCarga = capacidadCarga;
    }
    
    mostrarCapacidad() {
        return `Capacidad de carga: ${this.capacidadCarga} toneladas`;
    }
    
    tipoVehiculo() {
        return "Camión";
    }
}

// Crear instancias
const miCoche = new Coche("Toyota", "Corolla", 4);
const miMoto = new Moto("Honda", "CBR600", 600);
const miCamion = new Camion("Volvo", "FH16", 20);

// Utilizar métodos
console.log(miCoche.mostrarInfo());
console.log(miCoche.mostrarPuertas());
console.log(miMoto.mostrarCilindrada());
console.log(miCamion.mostrarCapacidad());

// Demostración de polimorfismo
const vehiculos = [miCoche, miMoto, miCamion];
vehiculos.forEach(vehiculo => {
    console.log(`${vehiculo.tipoVehiculo()}: ${vehiculo.mostrarInfo()}`);
});
